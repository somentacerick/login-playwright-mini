# Playwright Login Test Mini Project

This is a beginner-friendly QA automation project that tests a small login page using Playwright

## Project Purpose

The purpose of this project is to practice end to end testing by automating a realistic login workflow. The test suite checks both successful and unsuccessful login scenarios

## What This Project Tests

- Login page loads correctly
- Valid user can log in
- Wrong password is rejected
- Empty fields show validation
- Locked account cannot log in
- User can log out
